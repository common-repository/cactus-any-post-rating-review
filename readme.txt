== Description ==

Enable Rating & Review for any post type. 

3 rating styles supported: Point, Percentage and Star

Auto add rating to post and custom post types, or manual add using shortcode

This plugin is made and contributed by Power Elite Author <a href="https://www.cactusthemes.com">CactusThemes</a>
Demo page: <a href="http://unidash.cactusthemes.com/learn/">Unidash - University WordPress Theme</a>

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Rating Settings to configure the plugin
